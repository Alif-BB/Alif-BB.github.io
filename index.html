Things I've Built

Active Development · Personal Project

Automated Android Malware Triage Framework

A Python-based static analysis tool that automatically triages malicious APKs targeting Malaysian banking customers. Detects dangerous permissions, extracts hardcoded C2 IPs, Telegram bot tokens, and banking keywords using custom regex patterns. Risk scoring engine aggregates findings into CLEAN to CRITICAL classifications. Results delivered via a local Streamlit dashboard with one-click PDF forensic report export. Reduces manual triage time from 30-60 minutes to under 10 seconds.
